01 / THE ARCHITECTURE

A nervous system
for the helmet.

Axon distributes pressure, acceleration, and rotation sensors across the shell, streams them to a cloud backbone, and runs detection at the edge of every impact. The hardware feels the hit. The model decides what it means.

SENSE
Adafruit BNO085
A 9-DOF IMU in the helmet detecting sudden acceleration and rotational change.
PROCESS
Particle Boron MCU
Onboard microcontroller reading and processing the sensor data at the edge.
STREAM
Firebase Firestore
NoSQL cloud database ingesting flexible impact records in real time.
ALERT
FlutterFlow App
Coaches monitor player status live; athletes review post-activity statistics.
02 / SIGNAL PATH

From impact
to alert.

Four stages, one continuous loop, running every millisecond a player is on the field.

/01
Impact Captured
The BNO085 IMU registers sudden linear acceleration and rotational change inside the helmet.
/02
Telemetry Uplinked
The Particle Boron MCU streams readings to Firebase Firestore without leaving the sideline.
/03
Model Evaluates
Detection logic flags impacts that cross concussion-risk thresholds.
/04
Coach Notified
A real-time alert reaches the staff before a second hit can land.
